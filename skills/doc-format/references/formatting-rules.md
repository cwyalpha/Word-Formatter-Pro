# Formatting Rules

## 标题层级

| 类型 | 识别规则 | 默认字体 | 默认字号 |
|---|---|---|---:|
| 题目 | 文档开头连续居中且字体字号一致的段落；TXT/MD 首个非层级标题段落 | 方正小标宋简体 | 22 |
| 副标题 | 题目下方居中且字体字号不同的段落 | 楷体_GB2312 | 16 |
| 一级标题 | `一、`、`二、` 等 | 黑体 | 16 |
| 二级标题 | `（一）`、`（二）` 等 | 楷体_GB2312 | 16 |
| 三级标题 | `1.`、`2.` 等 | 仿宋_GB2312 | 16 |
| 四级标题 | `(1)`、`(2)` 等 | 仿宋_GB2312 | 16 |
| 正文 | 其他普通段落 | 仿宋_GB2312 | 16 |

## 页面和段落

- 页边距：上 3.7cm / 下 3.5cm / 左 2.8cm / 右 2.6cm
- 页脚距：2.5cm
- 正文行距：28 磅
- 题目/副标题行距：33 磅
- 页码：默认奇偶分页，奇数页右对齐，偶数页左对齐
- 页码字体：宋体，14pt
- 正文和标题会清理残留缩进，再设置标准首行缩进 2 字符
- `force_a4=false` 时 Word/WPS 文档保留原纸张大小；TXT/MD 默认使用 A4

## TXT/MD 特殊处理

- `.txt` 会按文本行生成段落
- `.md` 会清理 Markdown 标记，包括标题 `#`、粗体/斜体、链接、图片语法、引用、分隔线等
- 默认删除 TXT/MD 多余空行：单个空行删除，连续多个空行合并为一个
- 使用 `--keep-blank-lines` 可保留原始空行

## 表格、图表和附件

- 表格内容自动调整默认关闭，避免改变复杂表格布局
- 启用后可设置表头字体、表格字体、字号、行距、行高、表格宽度、边框、列宽和智能对齐
- 图/表标题通过图片或表格上下相邻的居中 `图...` / `表...` 段落识别
- 附件标识默认启用格式化，并会段前分页
